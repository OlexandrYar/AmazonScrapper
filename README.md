Just a Simple Flask Amazon web scraper that compares the prices across British, German and Irish websites with ability to save data in CSV

*How to use*
First clone or download the repo
```
git clone https://github.com/OlexandrYar/AmazonScrapper.git
```
After the repo was downloaded run 
```
pip install -r requirments.txt
```
then run app.py
```
python3 app.py
```
🚨Web Scraping could take a couple of minutes🚨
